# 📈 Stock Market Data Analyzer

A Python-based financial analytics project that analyzes stock market data using technical indicators, moving averages, volatility analysis, and RSI-based trading signals.

---

# 🚀 Features

- Real-time stock data fetching using Yahoo Finance
- Moving Average Analysis (SMA 20 & SMA 50)
- RSI Indicator Analysis
- Daily Returns Analysis
- Volatility Calculation
- Buy/Sell/Hold Signal Generation
- CSV Export
- Automated Report Generation
- Interactive Streamlit Dashboard

---

# 🛠️ Tech Stack

- Python
- Pandas
- NumPy
- Matplotlib
- yfinance
- ta
- Streamlit

# ▶️ How to Run

## Install dependencies

```bash
pip install -r requirements.txt
```

## Run Python Project

```bash
python main.py
```

## Run Streamlit Dashboard

```bash
streamlit run app.py
```
